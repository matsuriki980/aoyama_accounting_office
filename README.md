## 使用技術
- HTML5 / CSS3 (SCSS) / JavaScript
- CSSフレームワーク（Swiper）

## 構成
- index.html： トップページ
- feature.html： 青山会計事務所の強み
- service.html： サービス内容
- about.html： 事務所案内
- voice.html： お客様の声
- voice-detail.html： お客様の声 詳細 (全6ページ)
- news.html： お知らせ
- news-detail.html： お知らせ詳細 (全3ページ)
- privacy-policy.html： プライバシーポリシー
- contact： お問い合わせ
- assets/ : CSS・JS・画像
